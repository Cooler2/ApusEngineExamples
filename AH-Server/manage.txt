Управление сервером.

Сервер можно запускать как сервис либо как обычный процесс.

Параметры заруска
-install - инсталляция сервиса (имя сервиса берется из конфига)
-uninstall - удаление сервиса (имя установленного сервиса должно совпадать с именем в конфиге)
-run - запуск процесса (не сервиса!)

Управлять сервером можно редактируя файл конфига или записывая в файл command.txt команды

Запись в файл command.txt:
STOP - остановка (завершение процесса либо сервиса)
